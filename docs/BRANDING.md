# 品牌与命名建议

## 推荐主名称

Dragonfly Pulse

## 可选名称（按推荐顺序）

1. Dragonfly Pulse
- 品牌感强，短，记忆成本低

2. Dragonfly Campus Radar
- 含义清晰：校园 + 竞赛雷达
- 对外传播成本低

3. Dragonfly Insight Hub
- 适合后期从“竞赛”扩展到“校园机会”

## 中文名建议

蜻蜓校园赛情雷达

## 品牌故事（可直接用于宣发）

在自然界中，蜻蜓拥有由约 3 万个小眼组成的复眼结构，能实现全方位感知；
蜻蜓的神经传输速度极快，从发现目标到动作响应仅需极短时间。

Dragonfly Pulse 借用这两个特征：
- 全方位：多来源并行抓取与监控
- 低延迟：从公告发布到结构化入库的快速链路

## 一句话简介（推荐）

Dragonfly Pulse 是一个面向大学生的开源竞赛情报系统，帮助用户打破信息差，快速发现高价值科技赛事。
