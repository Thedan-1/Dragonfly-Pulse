# Release Checklist

## Must Have

- [ ] README is up to date
- [ ] LICENSE exists
- [ ] CONTRIBUTING, CODE_OF_CONDUCT, SECURITY exist
- [ ] .gitignore excludes local data and secrets
- [ ] CI workflow runs tests on pull requests

## Quality

- [ ] Tests pass locally
- [ ] scripts/run_daily.py finishes without crashing
- [ ] No hardcoded secrets
- [ ] .env is not tracked

## Open Source Readiness

- [ ] Replace placeholder security email in SECURITY.md
- [ ] Replace placeholder contact in CODE_OF_CONDUCT.md
- [ ] Add repository topics and description on GitHub
- [ ] Add issue templates and PR template (optional)
